
# `show-time.lp`

Handles show predicates which are generated during preprocessing for `#show` directives.
It includes the time argument in the show predicates, which is necessary for encodings that support time steps.

::: src/metasp/encodings/show-time.lp
    handler: asp
    options:
        encodings:
            git_link: true
        start_level: 1
