# Getting started

## Installation

=== "Pip"

```console
pip install metasp
```

=== "Development mode"

```console
git clone https://github.com/potassco/metasp.git/
cd metasp
pip install -e .[all]
```

!!! warning
    Use only for development purposes

## Usage

### Command line interface

Details about the command line usage can be found with:

```console
metasp -h
```
